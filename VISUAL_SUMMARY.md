## 🎯 FLUJO USUARIO COMPLETO

```
1. Usuario abre navegador
   └─ http://localhost:5173
   
2. Ve formulario de búsqueda
   ├─ Selecciona origen
   ├─ Selecciona destino
   ├─ Elige hora (opcional)
   └─ Elige preferencia
   
3. Presiona "Buscar Rutas"
   └─ Frontend envía POST al API
   
4. Backend procesa
   ├─ Valida estaciones
   ├─ Ejecuta Dijkstra
   ├─ Encuentra alternativas
   ├─ Calcula métricas
   ├─ Genera explicaciones
   └─ Rankea resultados
   
5. Frontend recibe respuesta
   ├─ Cachea en sessionStorage
   └─ Navega a resultados
   
6. Ve ruta recomendada
   ├─ Timeline visual
   ├─ Métricas detalladas
   └─ Explicación experta
   
7. Puede comparar rutas
   ├─ Selecciona hasta 3 rutas
   └─ Ve gráficos comparativos
   
8. Satisfecho con resultado
   ├─ Puede ver detalles
   ├─ Compartir ruta (futuro)
   └─ Descargar información (futuro)
```
